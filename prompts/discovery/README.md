# Discovery Prompts

Prompts for user research, problem framing, and hypothesis generation.

Use these when you're in the early stages of understanding a problem space — before you've committed to a solution.

---

## What Belongs Here

- User interview analysis and synthesis
- Problem framing and "how might we" generation
- Assumption mapping and hypothesis writing
- Jobs-to-be-done (JTBD) frameworks
- Persona development from raw research data
- Opportunity sizing and prioritization inputs

---

## How to Add a Prompt

1. Create a new `.md` file in this folder (e.g., `interview-synthesis.md`)
2. Follow the prompt writing guidelines in the [main README](../../README.md#-prompt-writing-guidelines)
3. Include: purpose, when to use it, the prompt itself, and a usage example
