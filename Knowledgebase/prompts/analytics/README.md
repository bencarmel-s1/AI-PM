# Analytics Prompts

Prompts for data interpretation, metric definitions, and OKR framing.

Use these when you need to make sense of data, define success, or communicate performance to stakeholders.

---

## What Belongs Here

- Metric definition and instrumentation planning
- OKR writing and alignment checks
- Data interpretation and insight extraction
- Dashboard design and KPI selection
- A/B test analysis and result framing
- Funnel analysis and drop-off identification

---

## How to Add a Prompt

1. Create a new `.md` file in this folder (e.g., `okr-writing.md`)
2. Follow the prompt writing guidelines in the [main README](../../README.md#-prompt-writing-guidelines)
3. Include: purpose, when to use it, the prompt itself, and a usage example

---

## Prompts in This Folder

| Prompt | What it does |
|---|---|
| [`okr-writing.md`](./okr-writing.md) | Write or refine OKRs from strategic goals and product context |

---

> **Tip:** When asking Claude to interpret data, paste in the raw numbers or chart values directly. The more specific the input, the more specific and actionable the output.
