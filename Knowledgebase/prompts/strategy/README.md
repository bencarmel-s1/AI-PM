# Strategy Prompts

Prompts for roadmapping, prioritization, and competitive analysis.

Use these when you're making decisions about *what* to build and *why* — translating insights into direction.

---

## What Belongs Here

- Roadmap scenario planning and trade-off analysis
- Prioritization frameworks (RICE, ICE, MoSCoW, etc.)
- Competitive landscape analysis and positioning
- Strategic narrative and "north star" framing
- OKR alignment and goal-setting inputs
- Build vs. buy vs. partner analysis

---

## How to Add a Prompt

1. Create a new `.md` file in this folder (e.g., `competitive-analysis.md`)
2. Follow the prompt writing guidelines in the [main README](../../README.md#-prompt-writing-guidelines)
3. Include: purpose, when to use it, the prompt itself, and a usage example
