# Template: Weekly Status Update

## Before you start

**What you need:** A quick brain dump of what you worked on this week. You can type it directly into Claude or save it as a file.

**Optional input:** Any task list, Jira export, or notes file. Save as `my-week.txt` and reference it.

**Tip:** If you keep a running notes file throughout the week, this workflow takes under 2 minutes on Friday afternoon.

---

## How to use
Type directly: `Write my weekly status update using @weekly-status-update.md. Here's what I worked on: [paste your notes]`

Or with a file: `Write my weekly status update using @weekly-status-update.md based on @my-week.txt`
