# Case Studies

Real examples of Claude Code used in our PM work.

This is one of the most valuable parts of this repo — seeing how teammates have actually used Claude Code in their day-to-day work is more useful than any template.

---

## What Belongs Here

- Real tasks you completed faster or better with Claude Code
- Experiments that didn't go as planned (and what you learned)
- Comparisons: doing a task manually vs. with Claude Code
- Surprising use cases you discovered

---

## How to Add a Case Study

Create a new `.md` file in this folder using the template below. Name it something descriptive (e.g., `prd-sprint-42.md`, `competitive-analysis-q1.md`).

---

## Case Study Template

```markdown
## [Task Name]
**PM:** [Your name]
**Date:** [YYYY-MM-DD]
**Time Saved:** [Estimate]

### The Challenge
What problem were you trying to solve?

### How Claude Code Helped
What did you ask it to do? What prompt/workflow did you use?

### Outcome
What was the result? What would you do differently?

### Prompt Used
[Paste the prompt here]
```
