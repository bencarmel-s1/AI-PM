# Ideas

Patterns, templates, and experiments — things we're testing, discussing, and evolving.

Browse freely. Try what looks useful. If something works well enough to become a habit, it belongs in the OS (`../os/`).

---

## What's Here

| Folder | What's there | Status |
|--------|-------------|--------|

| `advanced-workflow/` | Multi-step workflows for PRD creation, data analysis, product strategy, feature discovery, launch | Ready to use |

---

## How to Use

- **Starting out?** `basic-workflows/quickstart.md` is a 15-minute end-to-end walkthrough
- **Working on a PRD?** `advanced-workflow/prd-creation/` is the most mature workflow here
- **Want to understand a skill before running it?** `skills/` has the human-readable versions
- **Have a tested workflow to share?** Add it here and describe it — others will try it

## Contributing

If you've used something from here in real work, document the outcome in `case-studies/`. If you've improved a prompt or template, open a PR. If something from here has become part of your daily habit, propose moving it to the OS.
