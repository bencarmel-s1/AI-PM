## What It Covers

A hands-on course that teaches you how to use Claude Code for real PM work — not through videos or documentation, but by actually doing tasks inside Claude Code itself. The course spans 4 modules:

- **File operations & document management** — reading, writing, and organising project files
- **Parallel agent processing** — running multiple Claude instances simultaneously on complex tasks
- **Custom sub-agents & reviewers** — building specialised agents for specific PM workflows
- **Project memory & context management** — making Claude remember context across sessions
- **PRD writing with AI assistance** — drafting and refining product requirements documents
- **Data analysis for product decisions** — using Claude to interpret and summarise data
- **Competitive strategy development** — research and synthesis workflows
- **Image analysis & mockup feedback** — getting Claude to review designs and wireframes

---

## Who It's For

Product managers who want to integrate AI into their actual day-to-day work. No coding or terminal experience needed — just the ability to describe tasks in plain English and a basic PM background.

---

## Prerequisites

- Claude Pro or Max subscription
- Basic product management experience
- Mac, Windows, or Linux computer

## Why It's Worth Your Time

This course is directly relevant to what we're exploring in this repo. It's structured around real PM tasks (PRDs, data analysis, competitive research) and teaches you how to get Claude Code to do the heavy lifting. The interactive format means you're practising in the actual tool from lesson one — not watching someone else use it.
