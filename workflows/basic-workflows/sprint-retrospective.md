# Template: Sprint Retrospective

## Before you start

**What you need:** Nothing required upfront — you can describe the sprint verbally in your prompt, or paste in any notes you have.

**Optional input:** If your team uses a retro tool (Miro, FigJam, etc.), export or copy the sticky notes into a `.txt` file and reference it: `@retro-notes.txt`.

**Best used for:** Drafting a written retrospective after your live retro session, or structuring a solo async retro.

---

## How to use
`Run our sprint retrospective using @sprint-retrospective.md`

Or with notes: `Summarize @retro-notes.txt into a retrospective using @sprint-retrospective.md`
