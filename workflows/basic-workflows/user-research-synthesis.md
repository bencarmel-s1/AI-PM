# Template: User Research Synthesis

## Before you start

**What you need:** Each interview saved as a separate `.txt` or `.md` file, all in one folder (e.g. `user-interviews/interview-1.txt`, `interview-2.txt`, etc.).

**Where to put the files:** Create a subfolder called `user-interviews/` inside your working folder. Run `claude` from the parent folder.

**How to save interview notes:** Copy transcript or notes → paste into a text file → save with a descriptive name like `interview-sarah-march.txt`. Include the participant's role in the filename or at the top of the file.

**Minimum useful input:** 3+ interviews. For 1-2 interviews, ask Claude to summarize rather than synthesize.
