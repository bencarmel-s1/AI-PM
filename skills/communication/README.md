# Communication Skills

For drafting stakeholder-facing communications — when you need to write (not just review) leadership updates, difficult-news messages, or cross-functional announcements.

---

## Skills in This Folder

| Skill | What it helps you do |
|---|---|
| `stakeholder-communication.md` | Draft executive updates, bad-news framing, cross-functional announcements, and board prep |

---

## When to use these

When you need to create a first draft of any communication going to leadership, the board, or cross-functional stakeholders. Pairs with the `executive` agent for a complete draft -> review workflow.

---

## Example

```
@communication/stakeholder-communication.md
I need to send a leadership update about the Q2 launch delay.
Here are the sprint notes: @sprint-notes.md
And the updated timeline: @revised-timeline.md
Draft something honest but constructive — my VP needs to forward this to the CTO.
```
