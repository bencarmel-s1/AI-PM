# Claude Desktop — No-Terminal Alternative

Claude Desktop is the standalone app version of Claude. It includes a **Cowork** tab that lets Claude autonomously work with files on your computer, browse the web, and connect to tools like Gmail, Google Drive, and Notion — all without opening a terminal.

**When to use Claude Desktop (Cowork) vs Claude Code (terminal):**

| Task | Use |
|---|---|
| Organising files, renaming, creating spreadsheets | Claude Desktop / Cowork |
| Browsing the web and pulling data into documents | Claude Desktop / Cowork |
| Connecting to Gmail, Drive, or Notion | Claude Desktop / Cowork |
| Running PM templates with `@filename` | Claude Code (terminal) |
| Working with a CLAUDE.md context file | Claude Code (terminal) |
| Multi-document PM workflows (PRDs, research synthesis) | Claude Code (terminal) |
| Repo or code work | Claude Code (terminal) |

Both require a Claude Pro subscription ($17/month). Both use the same Claude models — Sonnet for everyday tasks, Opus for heavy lifting.
