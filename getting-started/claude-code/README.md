# Claude Code — Terminal Setup

Claude Code is the terminal version of Claude. It runs on your laptop and can read files directly from your folders, which is what powers the `@filename` pattern used in all the PM templates and workflows in this repo.

**When to use Claude Code vs Claude Desktop:**

| Task | Use |
|---|---|
| Running PM templates with `@filename` | Claude Code (terminal) |
| Working with a CLAUDE.md context file | Claude Code (terminal) |
| Multi-document PM workflows (PRDs, research synthesis) | Claude Code (terminal) |
| Repo or code work | Claude Code (terminal) |
| Organising files or creating spreadsheets autonomously | Claude Desktop / Cowork |
| Browsing the web or using Gmail/Drive connectors | Claude Desktop / Cowork |

Both require a Claude Pro subscription ($17/month).
